## 1. O QUE É O FREELANCEFLOW

**FreelanceFlow** é uma plataforma web para freelancers brasileiros criarem, enviarem e rastrearem propostas comerciais profissionais. O produto resolve o maior gargalo do trabalho autônomo: a etapa de fechar clientes.

**URL:** https://freelanceflow.com.br

**Slogan principal:**
> "Da proposta ao 'pode começar' — sem perder o timing."

**Taglines secundários:**
- "Crie propostas profissionais. Envie por e-mail ou WhatsApp. Saiba quando o cliente abriu."
- "Pare de perder clientes por falta de follow-up."
- "Sua proposta merece ser levada a sério."

---

## 2. O PROBLEMA QUE RESOLVE

Freelancers perdem contratos porque:
1. Fazem propostas no Word/Google Docs — sem profissionalismo visual
2. Enviam por email e nunca sabem se o cliente abriu
3. Esquecem de fazer follow-up — ou fazem tarde demais
4. Não têm controle de quais propostas estão abertas, aceitas ou perdidas
5. Perdem tempo recriando a mesma proposta do zero para cada cliente

**O FreelanceFlow resolve tudo isso em um único lugar.**

## 5. FUNCIONALIDADES PRINCIPAIS

### 5.1 Propostas Profissionais

**O que é:** Editor completo para criar propostas com visual profissional.

**Como funciona:**
- Usuário preenche: título, cliente, valor, prazo, condições de pagamento
- Adiciona seções dinâmicas conforme o projeto:
  - Descrição do serviço
  - Escopo detalhado (lista de itens)
  - Tabela de itens (descrição, quantidade, valor unitário)
  - Tabela de horas (perfil, horas, valor/hora)
  - Parcelas (descrição, porcentagem, condição)
  - Cláusulas contratuais
  - Timeline/cronograma com etapas
  - Imagens e referências visuais
  - O que está incluído (grid de checkmarks)
  - Tabela personalizada
- Gera PDF automaticamente com visual limpo e profissional
- Inclui logo, assinatura digital e cores da marca do freelancer

**Diferencial:** O PDF gerado parece feito por designer — não por Word.

---

### 5.2 Envio por E-mail

**O que é:** Envio direto da proposta para o cliente via email profissional.

**Como funciona:**
- Usuário clica "Enviar por e-mail"
- Preenche email e nome do destinatário
- Personaliza a mensagem (ou usa o texto padrão)
- Clica "Enviar"
- Cliente recebe email com PDF em anexo + botões de aprovação ("Aceitar" / "Recusar")
- Status da proposta muda para "Enviada"
- Evento registrado na timeline

---

### 5.3 Envio por WhatsApp

**O que é:** Envio da proposta via WhatsApp com link direto.

**Como funciona:**
- Usuário clica "Enviar por WhatsApp"
- Sistema gera mensagem pré-formatada com link único da proposta
- Usuário personaliza o nome do destinatário
- Clica "Abrir WhatsApp" → abre WhatsApp Web/App com mensagem pronta
- Cliente clica no link → abre a proposta no navegador
- Cliente pode aceitar ou recusar diretamente pela página pública

**Diferencial:** Funciona 100% no WhatsApp — canal favorito dos clientes brasileiros.

---

### 5.4 Rastreamento em Tempo Real

**O que é:** Sistema de notificação que avisa quando o cliente abre a proposta.

**Como funciona:**
- Cada proposta tem um link único com token de rastreamento
- Quando o cliente abre o link → evento "Visualizada" registrado instantaneamente
- Freelancer recebe notificação por email
- Dashboard mostra: "Visto X vezes" com datas e horários
- Status muda automaticamente: Enviada → Visualizada → Aceita/Recusada

**Informações rastreadas:**
- Quantas vezes foi aberta
- Em quais datas e horários
- Se o cliente aceitou ou recusou

---

### 5.5 Follow-up Automático

**O que é:** Sistema que lembra o freelancer (ou envia automaticamente) quando o cliente não respondeu.

**Como funciona:**
- 3 regras automáticas configuráveis:
  - **Regra 1:** Proposta não visualizada após X dias → lembrete
  - **Regra 2:** Proposta visualizada mas sem resposta após X dias → lembrete
  - **Regra 3:** Proposta expirando amanhã → aviso de urgência
- Central de follow-ups: lista de todos os lembretes agendados e enviados
- Possibilidade de adicionar lembretes manuais (via email ou WhatsApp)
- Configuração personalizável nas configurações

**Resultado:** Freelancer nunca mais perde um cliente por esquecimento.

---

### 5.6 CRM de Clientes

**O que é:** Cadastro e gestão de clientes com histórico completo.

**Como funciona:**
- Freelancer salva dados do cliente (nome, email, telefone)
- Cada cliente tem um perfil com:
  - Total de propostas enviadas
  - Propostas aceitas
  - Valor total em negócios fechados
  - Histórico completo de todas as propostas
- Acesso rápido para criar nova proposta para o mesmo cliente

**Plano Free:** Até 5 clientes
**Plano Pro:** Ilimitado

---

### 5.7 Modelos Prontos por Nicho (Pro)

**O que é:** Templates profissionais pré-estruturados por área de atuação.

**Nichos disponíveis:**
- Social Media
- Desenvolvimento de Website
- Design de Logo
- Fotografia
- Vídeo/Motion
- Consultoria
- (e outros)

**Como funciona:**
- Usuário Pro acessa a biblioteca de modelos
- Seleciona o nicho
- Clica "Usar modelo" → editor já carrega com a estrutura ideal
- Preenche apenas os dados do projeto e cliente

**Resultado:** Proposta profissional criada em menos de 5 minutos.

---

### 5.8 Dashboard com Métricas

**O que é:** Painel visual com indicadores de desempenho das propostas.

**Métricas exibidas:**
- Valor aprovado no mês (R$)
- Propostas em aberto
- Taxa de resposta (%)
- Total de propostas este mês
- Gráfico de barras: propostas por mês (histórico)
- Gráfico de rosca: distribuição por status (aceitas, recusadas, pendentes)
- Propostas recentes
- Propostas que precisam de atenção

## 6. JORNADA DO USUÁRIO (PASSO A PASSO)

### Passo 1 — Cadastro (30 segundos)
- Acessa freelanceflow.com.br
- Clica em "Começar grátis — sem cartão"
- Cadastra com email/senha ou Google
- Redireciona para o dashboard

### Passo 2 — Configuração do Perfil (2 minutos)
- Vai em Configurações
- Adiciona logo do negócio
- Preenche nome, documento, contato
- Escolhe assinatura digital (3 estilos de fonte)
- Define cor da marca para o PDF
- Adiciona redes sociais

### Passo 3 — Criar a Primeira Proposta (5-10 minutos)
- Clica em "+ Nova Proposta"
- Seleciona modelo (Pro) ou começa em branco
- Preenche: título, cliente, valor, prazo, condições
- Adiciona seções com o escopo do projeto
- Salva → PDF gerado automaticamente

### Passo 4 — Enviar para o Cliente (1 minuto)
- Clica "Enviar por e-mail" ou "Enviar por WhatsApp"
- Personaliza a mensagem
- Clica enviar / abre WhatsApp
- Status muda para "Enviada"

### Passo 5 — Acompanhar em Tempo Real
- Recebe notificação quando cliente abre
- Timeline mostra todos os eventos com horários
- Dashboard atualiza automaticamente

### Passo 6 — Follow-up (automático)
- Se cliente não responde → sistema agenda follow-up
- Freelancer é avisado ou o sistema envia automaticamente
- Central de follow-ups mostra todos os pendentes

### Passo 7 — Cliente Responde
- Cliente clica "Aceitar" ou "Recusar" na proposta
- Freelancer recebe email imediato com a resposta
- Status muda na plataforma
- Valor vai para "Valor Aprovado" no dashboard
